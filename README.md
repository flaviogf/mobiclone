## :computer: Techs

- [ASP.NET Core](https://docs.microsoft.com/pt-br/aspnet/?view=aspnetcore-3.1#pivot=core)

## :thinking: How to use

````sh
cd ./backend/Mobiclone
````

````sh
dotnet restore
````

````sh
cd ./backend/Mobiclone/Mobiclone.Api
````

````sh
dotnet ef database update
````

````sh
dotnet run
````
