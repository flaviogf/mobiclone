## :thinking: How to use

### Backend

````sh
cd ./backend/Mobiclone
````

````sh
dotnet restore
````

````sh
cd ./backend/Mobiclone/Mobiclone.Api
````

````sh
dotnet ef database update
````

````sh
dotnet run
````

### Mobile

````sh
cd ./mobile/mobiclone
````

````sh
flutter pub get
````

````sh
flutter run
````
